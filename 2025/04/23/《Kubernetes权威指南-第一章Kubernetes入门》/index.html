一、了解 Kubernetes

Kubernetes 基础知识：

​ Kubernetes 中，Service 是分布式集群架构的核心，一个 Service 对象具有以下特征：

  • 拥有唯一指定的名称（比如 mysql-server）
  • 拥有一个虚拟 IP 地址（ClusterIP）和端口号
  • 能够提供某种远程服务能力
  • 能够将客户端对服务的访问请求转发到一组容器应用上
