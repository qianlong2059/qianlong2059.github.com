Python __init__.py和import详解

Created2025-02-10|Updated2025-02-10|编程语言

__init__.py文件的作用是将文件夹变为一个Python模块，Python每个模块的包中的，都有__init__.py文件

Python模块和包的区别

  • 模块（Module）

在Python中，模块是一个包含Python代码的文件，通常以*.py为后缀。模块可以包含变量、函数、类等，并且可以通过import语句导入到其他程序中使用。例如，一个名为math.py的文件就是一个模块，可以通过import math*来使用其中的内容。

  • 包（Package）

包是一个包含多个模块的目录。包的目录中必须包含一个特殊的文件__init__.py，这个文件可以是空的，也可以包含初始化代码。包的结构类似于文件夹和文件的关系，一个包是一个目录，里面包含了一组Python模块文件和一个__init__.py文件。

模块和包的区别：

  • 结构：模块是一个独立的文件，而包是一个包含多个模块的目录
  • 导入方式：模块通过 import 语句导入，而包可以通过 import 包名.模块名 或 from 包名 import 模块名 的方式导入

__init__.py作用

通常__init__.py文件为空，但是可以为它增加其他的功能。在导入一个包时，实际上是导入了它的__init__.py文件。
